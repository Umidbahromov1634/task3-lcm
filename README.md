# Task 3 - LCM Service

Этот сервис разворачивает HTTP GET метод для нахождения НОК.

## URL

```
/umidbahromov24_gmail_com?x={}&y={}
```

## Примеры

- `?x=12&y=18` → `36`
- `?x=0&y=5` → `0`
- `?x=0&y=0` → `NaN`
- `?x=-3&y=7` → `NaN`
